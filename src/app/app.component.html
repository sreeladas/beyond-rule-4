About

This application is an open source side-project of a software developer chasing financial freedom. It is a minor modification on a fork of Beyond Rule 4 and was created to provide motivation and clarity on a path to financially independence.

This application doesn't collect or store any personal data before or after authorizing with YNAB. It may use browser cookies for browser-local session storage and essential application functionality (storing your YNAB authorization token) but does not use any analytics or tracking.

When the app authorizes with YNAB, it requests read-only access. It does not and cannot add or edit any information to your account or budget.

To revoke authorization at any time, visit https://app.ynab.com/settings, scroll to "Authorized Applications", and click "Revoke" for YFI. Thanks for trying it out!

YNAB OAuth

This application follows the Terms of Service for the YNAB API

  • The data obtained through the YNAB API is not stored on the application server. We may use browser local/session storage and/or cookies for application functionality, but API data is only received and used by the client browser.
  • Data obtained through the YNAB API will not unknowingly be passed to any third-party.
  • The application will never ask for any financial account credentials.
